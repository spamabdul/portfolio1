# Maaz Portfolio Website 🚀

## 📁 Project Structure

```
maaz/
├── index.html              # Main homepage
├── css/
│   └── style.css          # All styles
├── js/
│   └── script.js          # All JavaScript
├── assets/
│   └── images/            # Images folder
├── pages/
│   ├── about.html         # About page
│   ├── skills.html        # Skills page
│   ├── projects.html      # Projects page
│   └── contact.html       # Contact page
└── projects/
    ├── ecommerce/         # E-commerce project
    └── landing/           # Landing page project
```

## ✨ Features

- 🎨 **Professional Loading Animation** - Multi-layer animated loader
- 🖱️ **Custom Cursor** - Smooth following cursor with hover effects
- 💫 **Particle Background** - Animated particles canvas
- 📱 **Fully Responsive** - Works on all devices
- 🎭 **Smooth Animations** - CSS & JS animations throughout
- 🌈 **Glassmorphism Design** - Modern glass effects
- ⚡ **Fast Performance** - Optimized code

## 📄 Pages

1. **Home (index.html)** - Hero section with all sections
2. **About (pages/about.html)** - Detailed about section
3. **Skills (pages/skills.html)** - Skills and tools showcase
4. **Projects (pages/projects.html)** - Portfolio projects
5. **Contact (pages/contact.html)** - Contact form and info

## 🎯 Technologies Used

- HTML5
- CSS3 (Animations, Flexbox, Grid)
- JavaScript (ES6+)
- Font Awesome Icons
- Google Fonts (Inter, Space Grotesk)

## 🚀 How to Use

1. Open `index.html` in browser for homepage
2. Navigate to different pages using navbar
3. All pages are fully functional and independent

## 🎨 Color Scheme

- Primary: `#6c63ff` (Purple)
- Accent: `#00f5ff` (Cyan)
- Secondary: `#ff6584` (Pink)
- Background: `#0a0a0f` (Dark)

## 📱 Responsive Breakpoints

- Tablet: 768px - 1024px
